Ashley Hart is a Full-Stack Web Developer working to complete her learning at the University of Denver.

Since writing her first code in an online web development class through Florida State University, Ashley knew this was the professional path she was going to commit herself to.

While enrolled with the University of Denver for Full-Stack programming, Ashley is learning how to make beautiful websites using HTML, CSS, SASS, Javascript, and jQuery. Behind the scenes, she is learning to make these sites fun and functional with Node.js, APIs, AJAX, React.js and more.

While not online, Ashley can be found running trails in the high country, friendly competition at the CrossFit gym, and camping under the stars in the summer.
